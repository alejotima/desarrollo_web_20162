Manual de uso GIT

Como clonar un repo?
1- ubicarse en a carpeta destino
2- utilizar el comando (git clone)

Como realizar un commit en en repo?
1- utilizar el comando (git add .), para guardar los cambios realizados
2- utilizar el comando (git commit -m "(mensaje)"
3- utilizar el comando (git push -u origin master)

Como realizar una actualizacion de repo
1- ubicarse en a carpeta destino
2- utilizar el comando (git pull)
