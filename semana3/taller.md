# Taller # 1 de desarrollo web

Realizar las siguientes actividades:

1. Todo el grupo, se une para realizar una aplicación web que contenga lo mostrado en los siguientes PDF´s:
    * index.pdf : Debe ser la pagina inicial de la aplicacion (index.html)
        - NOTA: Alguno de ustedes debe hacer la estructura básica del index.html y cada uno adiciona el link respectivo a su hoja de vida
    * Hoja de vida.pdf : Cada uno de los estudiantes lo debe realizar, debe llamarse hv_ seguido de las iniciales del nombre de cada uno de ustedes. Ejemplo hv_mart.html
    
    El proyecto lo deben poner en la carpeta entrega de github de la siguiente manera:
    ```
    entrega
    └───index.html
    └───hv_mart.html    
    └───hv_test.html
    ```
    
2. Responder las siguientes preguntas y crear un archivo md (Markdown) escribiendo las respuestas:
    * ¿Por qué evitar las tablas de html?
    * ¿Qué es CSS?
    * ¿Donde se puede escribir CSS?
    * ¿Cuales son los selectores básicos en CSS?
    
    El el archivo Markdown lo deben poner en la carpeta entrega - markdown de github de la siguiente manera:
    ```
    entrega
    └───index.html
    └───hv_mart.html    
    └───hv_test.html
    └───markdown
        └───md_mart.md
        └───md_test.md
    ```
